# Lumière Beauty — Next.js 15 Website

## 🚀 Schnellstart

```bash
npm install
npm run dev
```

→ Öffne **http://localhost:3000**

## 🎨 Farben (in `app/globals.css` anpassen)
- Rosa `#ffa7a7` — Primärfarbe, CTAs
- Mint `#a7fffe` — Akzentfarbe
- Gold `#fff49d` — Pricing, Highlights

## 📂 Struktur
```
app/globals.css       ← Alle CSS-Variablen & Farben
app/layout.tsx        ← Root Layout
app/page.tsx          ← Hauptseite
components/           ← Alle Sektionen
```

## ✏️ Bilder ersetzen
Suchen Sie in den Komponenten nach `picsum.photos` und ersetzen Sie die URLs durch Ihre eigenen Fotos.

## 📦 Build
```bash
npm run build
npm run start
```

## 🔧 Stack
- Next.js 15 · React 19 · Tailwind CSS 3 · TypeScript
